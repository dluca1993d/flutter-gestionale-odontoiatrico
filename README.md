# App Gestionale Odontoiatrico

Progetto realizzato in **Flutter** per gestire appuntamenti, dati clinici e pazienti in uno studio odontoiatrico.

**Caratteristiche principali:**
- Gestione appuntamenti
- Archiviazione dati pazienti
- Interfaccia utente intuitiva
- Sviluppata in Flutter per dispositivi mobile

**Tecnologie utilizzate:** Flutter

**Obiettivo:** creare un'app pratica per la gestione di uno studio odontoiatrico, applicando conoscenze Flutter acquisite.
